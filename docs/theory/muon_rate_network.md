# Muon Catalytic Rate Network

## Populations

- $N_\mu$ — free muons
- $N_{d\mu}$ — muonic deuterium
- $N_{dt\mu}$ — muonic molecule
- $N_{(\alpha\mu)^+}$ — stuck muons

## Stripping

$$R_{strip} = R_{col} + R_{photon} + R_{proton}$$

$$\omega_s^{eff} = \omega_0 (1 - R_{col})(1 - R_X)$$

## Breakeven

$$N_{fus,\mu} > \frac{E_{muon}}{17.6 \text{ MeV}} \approx 284$$
